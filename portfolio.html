Viaja

Viaja, which means "go" or "travel" in Spanish, is a mobile application I completed as part of my dissertation project. The minimum viable product allows users to view their current location on a map, send & receive friend requests, "check-in" their location with friends, and view friends' check-ins on a social media feed.

Technologies

React Native | JavaScript | Node.js | Express.js | Cloud Firestore | Firebase Authentication | Google Maps API | Google Cloud Functions |

Link

https://github.com/madisonloehner/Viaja

I utilzed an array of new technologies and frameworks to develop this application, solidifying my interest in front-end development.
